<h2 id="project-description">Описание проекта</h2>

Проект представляет собой **веб-приложение**, которое помогает владельцам малого бизнеса решать повседневные задачи с
использованием **LLM (Language Learning Models)**.

Приложение предоставляет рекомендации и автоматизирует рутинные операции в таких сферах, как:

- Юридические вопросы
- Финансы и бухгалтерия
- Маркетинг и продвижение
- Операционные процессы и управление персоналом

Цель — **сэкономить время владельца бизнеса** и повысить качество ежедневных решений за счет контекстных подсказок и
проактивной аналитики.

---

<h2 id="tech-stack">Стек технологий</h2>

- Backend: **Python + FastAPI**
- Frontend: **React / Vite**
- База данных: **PostgreSQL**
- Контейнеризация: **Docker + Docker Compose**
- LLM: OpenRouter / OpenAI (выбор модели: `openai/gpt-oss-20b:free`)

<h2 id="local-setup">Локальный запуск</h2>

### 1. Перейдите в папку backend

```bash
cd src/backend
```

### 2. Создайте файлы окружения

- Создайте .env файл по примеру [`.env.example`](../src/backend/.env.example)
- Создайте .env.app_config файл по примеру [`.env.app_config.example`](../src/backend/.env.app_config.example)

### 3. Запустите Docker Desktop

### 4. Поднимите контейнеры через Docker Compose

```bash
docker compose --env-file .env up -d --build
```

### 5. Проверка работы сервиса

- **Backend**: откройте http://127.0.0.1:8000/docs
- **Frontend**: откройте http://127.0.0.1:80
